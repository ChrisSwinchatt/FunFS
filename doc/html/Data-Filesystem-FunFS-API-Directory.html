readDir :: Directory -> FS DirEntry #

Read the next directory entry from a directory. It is an error to attempt to read more than the number of entries in the directory, which can be checked with getDirSize.

mapDir :: (DirEntry -> FS a) -> Directory -> FS [a] #

Map a function over all the entries in a directory. NB: The reading position is not altered (unless the function argument to mapDir causes it) so thatthe code: entry1 <- readDir dir mapDir f dir entry2 <- readDir Has the same effect as if mapDir was removed, i.e. entry1 and entry2 are adjacent entries.

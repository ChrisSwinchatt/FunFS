data Handle #

A Filesystem handle - references an open file or directory within the filesystem. Essentially equivalent to POSIX - file descriptors (in fact it is exactly equal).

openHWithSize :: User -> Node -> OpenMode -> Int -> FS Handle #

Open a handle, overriding the size reported by the node. This is a workaround to allow directories to report their size as the number of entries they contain, rather than the sum of the sizes of the entries.

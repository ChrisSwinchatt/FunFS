A path to the file or device *in the *host* FS* which contains a filesystem.

The filesystem within IO. Use sync to use the filesystem in FS.

unmount :: FS () #

Unmount a filesystem. The unmount will fail if there are open handles.

forceUnmount :: FS () #

Unmount a filesystem forcibly. Invalidates all open handles.
